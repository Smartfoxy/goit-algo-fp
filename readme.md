# Моделювання кидання двох кубиків (Monte-Carlo)

## Мета роботи

Метою роботи є програмне моделювання кидання двох шестигранних ігрових кубиків,
побудова таблиці сум та їх імовірностей за допомогою методу Монте-Карло,
а також порівняння отриманих результатів з аналітичними розрахунками.

---

## Опис алгоритму

Алгоритм роботи програми складається з таких кроків:

1. Виконується велика кількість симуляцій кидання двох кубиків.
2. Для кожного кидка генерується випадкове число від 1 до 6 для кожного кубика.
3. Обчислюється сума значень, що випали на кубиках.
4. Підраховується кількість появ кожної можливої суми від 2 до 12.
5. Ймовірність кожної суми обчислюється як відношення кількості її появ до загальної кількості симуляцій.

---

## Аналітичні ймовірності

Аналітичні ймовірності для сум двох кубиків визначаються кількістю способів
отримати відповідну суму, поділеною на загальну кількість можливих комбінацій (36).

| Сума | Кількість способів | Ймовірність |
| ---: | -----------------: | ----------: |
|    2 |                  1 |      1 / 36 |
|    3 |                  2 |      2 / 36 |
|    4 |                  3 |      3 / 36 |
|    5 |                  4 |      4 / 36 |
|    6 |                  5 |      5 / 36 |
|    7 |                  6 |      6 / 36 |
|    8 |                  5 |      5 / 36 |
|    9 |                  4 |      4 / 36 |
|   10 |                  3 |      3 / 36 |
|   11 |                  2 |      2 / 36 |
|   12 |                  1 |      1 / 36 |

---

## Порівняння результатів

Для порівняння результатів методу Монте-Карло з аналітичними значеннями
було проведено експерименти з різною кількістю симуляцій:

- `n = 36`
- `n = 1_000`
- `n = 10_000`
- `n = 100_000`

Для кожного значення `n` виводиться таблиця, що містить:

- кількість появ кожної суми;
- Monte-Carlo ймовірність;
- аналітичну ймовірність;
- абсолютну різницю між ними.

---

## Графічне представлення результатів

Для наочного аналізу результатів побудовано стовпчиковий графік,
який відображає порівняння Monte-Carlo ймовірностей та аналітичних значень
для кожної можливої суми.

---

## Висновки

- При малій кількості симуляцій результати методу Монте-Карло можуть
  суттєво відрізнятися від аналітичних значень через випадковий характер процесу.
- Зі збільшенням кількості симуляцій результати поступово збігаються
  з аналітичними ймовірностями.
- Найбільшу ймовірність має сума **7**, що підтверджується як аналітичними
  розрахунками, так і результатами симуляції.
- Отримані результати підтверджують коректність реалізації алгоритму
  та демонструють ефективність методу Монте-Карло для статистичного моделювання.
